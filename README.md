# upy_survey
A web-app with the purpose of making surverys at the UPY.
Before to start, you should enter some commands in GitBash which are:
1. "npm install"
2. "-g nodemon"
3. "nodemon"

After all of this, you should type in your webbrowser "localhost:3000" and you will have acces to our very first beta.
After this enter "test@upy.edu.mx"  in the uset and in password "test".
Answer as a common survey.
